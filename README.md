# star-ui

## Project setup
```
npm install
```

### Compiles and hot-reloads for development (for doc and test)
```
npm run serve
```

### Compiles and minifies for production (for doc)
```
npm run build
```

### Compiles and minifies for production (for ui only)
```
npm run build-ui
```

### Lints and fixes files
```
npm run lint
```
